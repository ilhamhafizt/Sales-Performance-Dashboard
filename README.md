# 📊 Sales Performance Dashboard

## 📌 Overview

This project is a complete end-to-end data analysis workflow that demonstrates the process of transforming raw sales data into meaningful insights using **Python, SQL, and Data Visualization**.

The project includes:

* Data cleaning using Python
* Data analysis using SQL
* Interactive visualization using HTML, CSS, and JavaScript

---

## 🧰 Tech Stack

* 🐍 Python (Data Cleaning)
* 🗄️ SQL (Data Analysis)
* 🌐 HTML, CSS, JavaScript (Dashboard Visualization)
* 📁 CSV Dataset

## ⚙️ Data Cleaning Process (Python)

The raw dataset is processed using Python to:

* Handle missing values
* Remove duplicates
* Standardize data formats
* Prepare clean dataset for analysis

Run:

```bash
python data_cleaning.py
```

---

## 🗄️ Data Analysis (SQL)

SQL is used to extract insights such as:

* Total sales
* Best-selling products
* Revenue trends
* Customer behavior

Queries are available in:

```
sql_analysis/sales_analysis.sql
```
